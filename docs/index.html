Toon Agent Bridge

Universal bridge for converting OpenAPI and agents.json schemas into agent-ready workflows

Quick Start

Install the packages you need:

npm install @toon/core @toon/backend-node @toon/frontend

Create a server from your OpenAPI schema:

import { createServer } from '@toon/backend-node'; const server = await createServer({ port: 3000, schemaSource: './openapi.json', cors: true, }); await server.start();

Use React components for visualization:

import { AgentVisualizer } from '@toon/frontend'; import '@toon/frontend/styles'; <AgentVisualizer url="http://localhost:3000/events" protocol="sse" autoConnect={true} />

Packages

@toon/core

Core types, schema parsers, and normalization logic.

npm install @toon/core

@toon/backend-node

Node.js/TypeScript backend adapter with Fastify, SSE, and WebSocket.

npm install @toon/backend-node

@toon/frontend

React components for agent visualization.

npm install @toon/frontend

toon-backend-python

Python backend adapter with FastAPI, SSE, and WebSocket.

pip install toon-backend-python
