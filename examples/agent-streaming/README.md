# Agent Streaming Example

Agent streaming with TOON format using @toon/backend-node and @toon/frontend.

## Setup

```bash
cd examples/agent-streaming
pnpm install
pnpm start
```

## Features

- TOON format streaming endpoint
- Real-time event streaming
- Token counting
- Format conversion API

## Usage

1. Start the server: `pnpm start`
2. Access streaming endpoint: `GET /stream/toon?file=data.json`
3. Use frontend components to visualize streaming data
