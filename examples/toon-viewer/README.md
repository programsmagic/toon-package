# TOON Viewer Example

Standalone TOON format viewer application.

## Setup

```bash
cd examples/toon-viewer
pnpm install
pnpm dev
```

## Features

- Drag & drop file support
- Real-time format conversion
- Token counting
- LLM-grade mode (copy as fenced block)
- Split-pane viewer

## Usage

1. Start the dev server: `pnpm dev`
2. Open the app in your browser
3. Drag & drop a JSON, TOON, CSV, or YAML file
4. View and convert between formats
5. Copy optimized format for LLM prompts
